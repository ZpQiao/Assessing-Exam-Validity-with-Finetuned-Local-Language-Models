# Only-Text Exam Data

This folder holds the intermediate exam data after OCR (Nougat or Mathpix) and
the parsed per-exam JSON files.

---

## Files

- `exam_*.txt`  
  Raw text exports of each exam (one file per PDF).

- `exam_*.json`  
  Parsed questions for a single exam (one JSON file per exam).

- `merged/`  
  Optionally contains cleaned/merged versions of the per-exam JSON files.

- `all_questions.json`  
  Concatenation of all per-exam JSON files, used as the main input for later
  stages.

- `under_six_options.csv`, `over_six_options.csv`, `blank_options_report.csv`  
  CSV reports generated by the statistics subcommands of `qa_toolkit.py`.

- `qa_toolkit.py`  
  Main CLI tool for this folder.
